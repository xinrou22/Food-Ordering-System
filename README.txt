---------------------------------------- User Manual --------------------------------------------------
                    
Step 1 : Extract the file
Step 2 : Copy the main project folder (FoodOrderingSystem).
Step 3 : Paste it in xampp/htdocs/
Step 4 : Open the browser and go to URL --> http://localhost/phpmyadmin/
Step 5 : Click on the database tab.
Step 6 : Create a database named "fos_db" and then click on the import tab.
Step 7 : Click on "Choose File" and select the "fos_db.sql" which is available inside the "FoodOrderingSystem" folder.
Step 8 : Click on go.
Step 9 : Open a browser and go to URL --> http://localhost/FoodOrderingSystem/
         For Backend Admin Panel, go to URL --> http://localhost/FoodOrderingSystem/admin/

There you go, enjoy!
